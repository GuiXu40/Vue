#### :hearts:为什么使用组件
使代码复用
#### :hearts:组件用法
组件与创建Vue实例的方法类似,需要注册之后才能使用
+ 全局注册: 任何Vue实例都可使用
```JavaScript
Vue.component('my-component',{
  //选项
})
```
要在父实例中使用这个组件,必须要在实例创建前注册
```JavaScript
<div id="app">
    <my-component></my-component>
</div>
<script>
  Vue.component('my-component',{
      //选项
  })
  
  var app=new Vue({
      el: '#app'
  });
</script>
```
组件选项中添加template就可以显示组件内容,**template的DOM结构必须被一个元素包含**
```JavaScript
  Vue.component('my-component',{
      template: '<div>这是组件的内容</div>'
  })
```
+ 局部注册
在Vue的实例中,可以使用components选项局部注册组件,只在该实例作用域下有效,组件中也可以使用components选项来注册组件,使组件可以嵌套
```JavaScript
<body>
    <div id="app">
        <my-component></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var Child={
        template: '<div>这是组件的内容</div>'
    }
    var app=new Vue({
        el: '#app',
        components: {
            'my-component': Child
        }
    });
</script>
```
Vue组件的模板在某些情况下会受到HTML的限制,比如table内只能是表格元素,这种情况下,可以使用特殊属性来挂载组件
```JavaScript
<body>
    <div id="app">
        <table>
            <tbody is="my-component"></tbody>
        </table>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div>这里是组件的内容</div>'
    })
    var app=new Vue({
        el: '#app',
    });
</script>
```
如果使用的是字符串模板,是不受限制的

除了template选项外,组件中也可以使用Vue实例中那个样的选项,比如data,但是data必须是函数,然后将数据return出
```JavaScript
<body>
    <div id="app">
        <my-component></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div>{{message}}</div>',
        data: function(){
            return{
                message: '组件内容'
            }
        }
    })
    var app=new Vue({
        el: '#app',
    });
</script>
```
JavaScript对象是引用关系,所以如果return出的对象引用了外部的一个对象,那么这个对象就是共享的,任何一方修改都会同步,实例
```JavaScript
<body>
    <div id="app">
        <my-component></my-component>
        <my-component></my-component>
        <my-component></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var data={
        counter: 0
    };
    Vue.component('my-component',{
        template: '<button @click="counter++">{{counter}}</bottun>',
        data: function(){
            return data;
        }
    });
    var app=new Vue({
        el: '#app',
    });
</script>
```
点击任何一个bottun,3个数字都会加1,那是因为data引用的是外部的对象,相反的效果
```JavaScript
<body>
    <div id="app">
        <my-component></my-component>
        <my-component></my-component>
        <my-component></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<button @click="counter++">{{counter}}</bottun>',
        data: function(){
            return {
                counter: 0
            }
        }
    });
    var app=new Vue({
        el: '#app',
    });
</script>
```

## :egg:使用props传输数据
:spades:<a href="#title">回到目录</a><br>
#### :hearts:基本用法
组件之间可以通信,通常父组件的模板中包含子组件,父组件要正向的向子组件中传递数据或参数,子组件接收到后根据参数的不同来渲染不同的内容或执行操作,这个正向传递就是通过props来实现
```JavaScript
<body>
    <div id="app">
        <my-component message="来自父组件的数据"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div>{{message}}</div>',
        props: ['message']
    });
    var app=new Vue({
        el: '#app',
    });
</script>
```
使用v-bind可以实现动态绑定父级的数据  绑定props的数据
```JavaScript
<body>
    <div id="app">
        <input type="text" v-model="parentMessage">
        <my-component :message="parentMessage"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div>{{message}}</div>',
        props: ['message']
    });
    var app=new Vue({
        el: '#app',
        data: {
            parentMessage: ''
        }
    });
</script>
```
**如果你直接传递数字,布尔值,数组,对象,而不使用v-bind,传递的数据只是字符串**
```JavaScript
<body>
    <div id="app">
        <my-component message="[1,2,3]"></my-component>
        <my-component :message="[1,2,3]"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div>{{message.length}}</div>',
        props: ['message']
    });
    var app=new Vue({
        el: '#app'
    });
</script>

7,3
```
#### :hearts:单向数据流
单向数据: 顾名思义,就是父级可以传递数据给子组件,但子组件不可以传出来

两种需要改变 prop的情况:
+ 父组件传递初始值进来,子组件将他作为初始值保存起来,在自己的作用域下可以随意使用和修改,这种情况下可以在组件data内在声明一个数据,引用父组件的prop
```JavaScript
<body>
    <div id="app">
        <my-component :init-count="1"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div>{{count}}</div>',
        props: ['initCount'],
        data: function(){
            return {
                count: this.initCount
            }
        }
    });
    var app=new Vue({
        el: '#app'
    });
</script>
```
+ 第二种情况就是prop作为需要被转变的原始值传入,这种情况使用计算属性就可以
```JavaScript
<body>
    <div id="app">
        <my-component :width="100"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<div :style="style">组件内容</div>',
        props: ['width'],
        computed: {
            style: function(){
                width: this.width+'px'
            }
        }
    });
    var app=new Vue({
        el: '#app'
    });
</script>
```
**注意**: 在JavaScript中对象和数组是引用对象,指向同一个内存空间,所以props是对象和数组时,在子组件内改变会影响父组件的

## :egg:组件通信

#### :hearts:自定义事件
当子组件需要向父组件传递数据时,就要用到自定义事件,子组件使用$emit()来触发事件,父组件用$on()来监听子组件的时间,

父组件也可以直接在子组件的自定义标签上使用v-on来监听子组件触发的自定义事件
```JavaScript
<body>
    <div id="app">
        <p>总数:{{total}}</p>
        <my-component @increase="handleGetTotal" @reduce="handleGetTotal"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '\<div>\<button @click="handleIncrease">+1</button>\<button @click="handleReduve">-1</button>\</div>',
        data: function(){
            return {
                counter: 0
            }
        },
        methods: {
            handleIncrease: function(){
                this.counter++;
                this.$emit('increase',this.counter);
            },
            handleReduve: function(){
                this.counter--;
                this.$emit('reduce',this.counter);
            }
        }
    });
    var app=new Vue({
        el: '#app',
        data: {
            total: 0
        },
        methods: {
            handleGetTotal: function(total){
                this.total=total;
            }
        }
    });
</script>
```
除了监听自定义事件外,也可以监听DOM事件,可以用.native修饰符表示监听的是一个原生事件,监听的是该组件的根元素(完全没有理解)
```JavaScript
<my-component v-on:click.native="handleClick"></my-component>
```
#### :hearts:使用v-model
```JavaScript
<body>
    <div id="app">
        <p>总数:{{total}}</p>
        <my-component v-model="total"></my-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        template: '<button @click="handleClick">+1</button>',
        data: function(){
            return {
                counter: 0
            }
        },
        methods: {
            handleClick: function(){
                this.counter++;
                this.$emit('input',this.counter);
            }
        }
    });
    var app=new Vue({
        el: '#app',
        data: {
            total: 0
        }
    });
</script>
```
v-model还可以用来创建自定义的表单输入组件,进行数据的双向绑定
```JavaScript
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    <div id="app">
        <p>总数:{{total}}</p>
        <my-component v-model="total"></my-component>
        <button @click="handleReduce">-1</button>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    Vue.component('my-component',{
        props: ['value'],
        template: '<input :value="value" @input="updateValue">',
        methods: {
            updateValue: function(event){
                this.$emit('input',event.target.value);
            }
        }
    });
    var app=new Vue({
        el: '#app',
        data: {
            total: 0
        },
        methods: {
            handleReduce: function(){
                this.total--;
            }
        }
    });
</script>
```
实现一个具有双向数据绑定的v-model组件要满足下面两个要求
+ 接收一个value属性
+ 在有新的value时触发input事件
#### :hearts:非父子组件通信
使用一个中间媒介来达到信息交互的目的
```JavaScript
<body>
    <div id="app">
        {{message}}
        <component-a></component-a>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var bus=new Vue();
    Vue.component('component-a',{
        template: '<button @click="handleEvent">传递事件</button>',
        methods: {
            handleEvent: function(){
                bus.$emit('on-message','来自组件的内容')
            }
        }
    });
    var app=new Vue({
        el: '#app',
        data: {
            message: ''
        },
        mounted: function(){
            var _this=this;
            bus.$on('on-message',function(msg){
                _this.message=msg;
            });
        }
    });
</script>
```
除了此方法,还有两种可以实现组件间通信: 父链和子组件索引

## :egg:使用slot分布内容
:spades:<a href="#title">回到目录</a><br>
#### :hearts:什么是slot
当需要让组件使用,混合父组件的内容与子组件的模板时,就会用到slot-->这个过程叫做内容分发
#### :hearts:作用域
```JavaScript
<child-component>
    {{message}}
</child-component>
```
此时的message就是一个slot,是**父组件的数据**
#### :hearts:slot用法
+ 单个slot
在子组件内使用特殊slot元素就可以为这个子组件开启一个slot,在父组件模板里,插入在子组件标签内的内容将替代子组件的slot标签内的内容
```JavaScript
<body>
    <div id="app">
        <child-component>
            <P>分发的内容</P>
            <P>更多分发的内容</P>
        </child-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var bus=new Vue();
    Vue.component('child-component',{
        template: '\
            <div>\
                <slot>\
                    <p>如果父组件没有插入内容,我将默认出现</p>\
                </slot>\
            </div>\
        '
    });
    var app=new Vue({
        el: '#app'
    });
</script>
```
**注意**: 子组件slot内的备用内容,他的作用域是子组件本身
+ 具名slot
给slot元素指定一个name后可以分发多个内容,具名slot可以与单个slot共存
```JavaScript
<body>
    <div id="app">
        <child-component>
            <h1 slot="header">标题</h1>
            <p>正文部分</p>
            <p>更多的正文内容</p>
            <div slot="footer">底部信息</div>
        </child-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var bus=new Vue();
    Vue.component('child-component',{
        template: '\
            <div class="container">\
                <div class="header">\
                    <slot name="header"></slot>\
                </div>\
                <div class="main">\
                    <slot></slot>\
                </div>\
                <dvi class="footer">\
                    <slot name="footer"></slot>\
                </div>\
            </div>\
        '
    });
    var app=new Vue({
        el: '#app'
    });
</script>
```
#### :hearts:作用域插槽
使用一个可以复用的模板替换已渲染元素

例:
```JavaScript
<body>
    <div id="app">
        <child-component>
            <template scope="props">
                <p>来自父组件的内容</p>
                <p>{{props.msg}}</p>
            </template>
        </child-component>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var bus=new Vue();
    Vue.component('child-component',{
        template: '\
            <div class="container">\
                <slot msg="来自子组件的内容"></solt>\
            </div>\
        '
    });
    var app=new Vue({
        el: '#app'
    });
</script>
```
作用域插槽允许组件自定义应该如何渲染列表每一项
```JavaScript
<body>
    <div id="app">
        <my-list :books="books">
            <template slot="book" scope="props">
                <li>{{props.bookName}}</li>
            </template>
        </my-list>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var bus=new Vue();
    Vue.component('my-list',{
        template: '\
            <ul>\
                <slot name="book"\ v-for="book in books"\ :book-name="book.name">\</slot>\
            </ul>\
        ',
        props: {
            books: {
                type: Array,
                default: function(){
                    return [];
                }
            }
        }
    });
    var app=new Vue({
        el: '#app',
        data: {
            books: [
                {name: 'afda'},
                {name: 'acccc'},
                {name: 'ggggg'}
            ]
        }
    });
</script>
```
