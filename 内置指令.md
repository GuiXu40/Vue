## :egg:基本指令
:spades:<a href="#title">回到目录</a><br>
#### :hearts:v-cloak
不需要表带式,作用是:他会在Vue实例结束编译时从绑定的HTML元素上移除,通常和css的display: none;配合使用
```JavaScript
[v-cloak] {
    display: none;
}
```
#### :hearts:v-once
也是一个不需要表达式的指令,意思是他的元素或组件只渲染一次,包括元素或组件的所有子节点.首次渲染之后,不在随数据的变化重新渲染,被视为静态页面
<p id="p2"></p>

## :egg:条件渲染指令
:spades:<a href="#title">回到目录</a><br>
#### :hearts:v-if,v-else-if,v-else
可以根据表达式的值在DOM中渲染或销毁元素/组件
```JavaScript
<body>
    <div id="app" v-cloak>
        <p v-if="status===1">当status为1时显示该行</p>
        <p v-else-if="status===2">当status为1时显示该行</p>
        <p v-else>否则显示该行</p>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    new Vue(
        {
            el: '#app',
            data: {
                status: 1
            }
        }
    )
</script>
```
如果一次判断的是多个元素,可以使用Vue.js内置的template元素上使用条件命令
```JavaScript
<body>
    <div id="app">
        <template v-if="status===1">
            <p>这是一段文本</p>
            <p>这是一段文本</p>
            <p>这是一段文本</p>
        </template>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    new Vue(
        {
            el: '#app',
            data: {
                status: 1
            }
        }
    )
</script>
```
Vue在渲染元素时,处于效率考虑,会尽可能的覆用已有的元素而非重新渲染
```JavaScript
<body>
    <div id="app">
        <template v-if="type==='name'">
            <label>用户名:</label>
            <input type="text" placeholder="输入用户名">
        </template>
        <template v-else>
            <label>邮箱:</label>
            <input type="text" placeholder="输入用户名">
        </template>
        <button @click="handle">切换输入类型</button>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    new Vue(
        {
            el: '#app',
            data: {
                type: 'name'
            },
            methods: {
                handle: function(){
                    this.type=this.type==='name'?'mail':'name';
                }
            }
        }
    )
</script>
```
key属性可以取消复原
#### :hearts:v-show
v-show的用法与v-if基本一致,但本质上是改变元素的display,当v-show表达式的值为false时,元素会隐藏
```JavaScript
<body>
    <div id="app">
        <p v-show="status===1">当status为1时显示该行</p>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    new Vue(
        {
            el: '#app',
            data: {
                status: 2
            }
        }
    )
</script>
```
#### :hearts:v-if与v-show的选择
v-if是条件渲染,会根据表达式适当地销毁或重建元素及绑定的事件或子组件

v-show只是简单的css属性切换,无论条件是否为真,都会被编译,相比之下,v-if更适合条件不经常改变的场景

v-show适用于频繁切换条件

## :egg:列表渲染指令v-for
:spades:<a href="#title">回到目录</a><br>
#### :hearts:基本用法
遍历数组或者循环对象
```JavaScript
<body>
    <div id="app">
        <ul>
            <li v-for="book in books">{{book.name}}</li>
        </ul>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    new Vue(
        {
            el: '#app',
            data: {
                books: [
                    {name: 'Vue'},
                    {name: 'java'},
                    {name: 'c++'}
                ]
            }
        }
    )
</script>
```
也可以使用of代替in

v-for的表达式支持一个可选参数作为当前项的索引
```JavaScript
<body>
    <div id="app">
        <ul>
            <li v-for="(book, index) in books">{{index}}--{{book.name}}</li>
        </ul>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    new Vue(
        {
            el: '#app',
            data: {
                books: [
                    {name: 'Vue'},
                    {name: 'java'},
                    {name: 'c++'}
                ]
            }
        }
    )
</script>
```
v-for也可以用在template标签中,将多个元素进行渲染
```JavaScript
<body>
    <div id="app">
        <ul>
            <template v-for="book in books">
                <li>书名: {{book.name}}</li>
                <li>作者: {{book.author}}</li>
            </template>
        </ul>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>

        {
            el: '#app',
            data: {

                    {name: 'java',author: 'guixu'},
                    {name: 'c++',author: 'guixu'}

            }
        }
    )
</script>
```
除了数组外,对象的属性饿死可以遍历的

#### :hearts:数组更新
Vue包含了一组观察数组变异的方法,使用他们改变数组会触发视图更新

会改变原始的数组的方法:
+ push() 
+ pop()
+ shift()
+ unshift()
+ splice()
+ sort()
+ reverse()
```JavaScript
<body>
    <div id="app">
        <ul>
            <template v-for="book in books">
                <li>书名: {{book.name}}</li>
                <li>作者: {{book.author}}</li>
            </template>
        </ul>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var app=new Vue(
        {
            el: '#app',
            data: {
                books: [
                    {name: 'Vue',author: 'guixu'},
                    {name: 'java',author: 'guixu'},
                    {name: 'c++',author: 'guixu'}
                ]
            }
        }
    )
        app.books.push({
            name: 'css',
            author: 'guixu'
        });
</script>
```
不改变原数组的方法:返回一个新数组
+ filter()
+ concat()
+ slice()
```JavaScript
<body>
    <div id="app">
        <ul>
            <template v-for="book in books">
                <li>书名: {{book.name}}</li>
                <li>作者: {{book.author}}</li>
            </template>
        </ul>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var app=new Vue(
        {
            el: '#app',
            data: {
                books: [
                    {name: 'Vue',author: 'guixu'},
                    {name: 'aajava',author: 'guixu'},

            }

```

+ 修改数组长度

```JavaScript
    name: 'asdf',
    author: 'guixu'
})
```
+ 使用splice
#### :hearts:过滤与排序
当你不想改变原数组,想通过一个数组的副本来做过滤或排序的显示时,可以使用计算属性来返回过滤或排序后的数组
```javascript
<body>
    <div id="app">
        <ul>
            <template v-for="book in filterbooks">
                <li>书名: {{book.name}}</li>
                <li>作者: {{book.author}}</li>
            </template>
        </ul>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var app=new Vue(
        {
            el: '#app',
            data: {
                books: [
                    {name: 'Vue',author: 'guixu'},
                    {name: 'aajava',author: 'guixu'},
                    {name: 'aac++',author: 'guixu'}
                ]
            },
            computed: {
                filterbooks: function(){
                    return this.books.filter(function(book){
                        return book.name.match(/aa/);
                    });
                }
            }
        }
    )
</script>
```

## :egg:方法与事件
:spades:<a href="#title">回到目录</a><br>
#### :hearts:基本用法
```javascript
<body>
    <div id="app">
        点击次数: {{counter}}
        <button @click="counter++">+1</button>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var app=new Vue(
        {
            el: '#app',
            data: {
                counter: 0
            }
        }
    )
</script>
```
@click的表达式可以直接使用JavaScript语法,也可以是一个methods中的方法
```JavaScript
<body>
    <div id="app">
        点击次数: {{counter}}
        <button @click="handleAdd()">+1</button>
        <button @click="handleAdd(10)">+10</button>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var app=new Vue(
        {
            el: '#app',
            data: {
                counter: 0
            },
            methods: {
                handleAdd: function(count){
                    count=count || 1;
                    this.counter+=count;
                }
            }
        }
    )
</script>
```
@click调用的方法后可以不跟括号"()"

vue提供了一个特殊变量$event,用于访问DOM事件,实例: 阻止连接打开
```JavaScript
<body>
    <div id="app">
        <a href="https://www.baidu.com" @click="handleAdd('禁止打开',$event)">打开链接</a>
    </div>
</body>
<script src="js/vue.min.js"></script>
<script>
    var app=new Vue(
        {
            el: '#app',
            data: {
                counter: 0
            },
            methods: {
                handleAdd: function(message,event){
                    event.preventDefault();
                    window.alert(message);
                }
            }
        }
    )
</script>
```
